A custom WordPress plugin created with Dyad.

== Description ==

This is a sample WordPress plugin that demonstrates best practices for plugin development. It includes:

* Custom post types and taxonomies
* Admin settings page
* AJAX functionality
* Shortcodes
* Proper security practices
* Translation ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Custom Plugin screen to configure the plugin

== Frequently Asked Questions ==

= How do I use the shortcode? =

Use `[custom_plugin_hello name="Your Name"]` in any post or page.

= Is this plugin translation ready? =

Yes, the plugin is fully translation ready and includes a POT file.
